## Why this matters

The `image = "0.25"` dependency is listed in `Cargo.toml` but never imported anywhere in the source tree (`use image`, `image::`, or `extern crate image` appear zero times in src/ and galaxy-shader/). Unused dependencies increase compile time (the `image` crate has many features and transitive deps), increase binary size (dead code elimination may not catch everything), and add noise to `cargo update`/`cargo audit` output. Removing it is a one-line change.

The `image` crate was likely intended for a future PNG export feature — the repo contains a `galaxy.png` reference render — but since it's unused now, it should be removed and re-added when that feature is actually implemented.
